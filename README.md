# 第 16 天：Svelte 中的邏輯運作：`each` 邏輯區塊

> 「我的 `each` 邏輯區塊看起來真棒，不是嗎？」他徵詢我的意見。「看看它是如何將繁瑣的工作處理得如此簡潔漂亮。」
> 我同意這個 `each` 邏輯區塊的確厲害。
> 「沒錯。」他掃視著那些 `each` 邏輯區塊，從每一個開頭到每一個結尾，「我花了整整三年時間掙得的錢才賣下了它。」
>
> ～節錄自《The Great Svelte：第五章》
